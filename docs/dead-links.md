## First: why this has to be done in the slides (and we can't just fix the website for you)

Your Google Slides deck is the **master copy.** The website is built *from* your slides automatically — think of it like the website is a photocopy of your deck. Every time we rebuild the site, it copies all the links straight out of your slides again, fresh.

So if we quietly fixed a link only on the website, the **next** time you change anything in your deck and we rebuild, the website would copy your slides again — and the old broken link would come right back. The fix would disappear.

**The only place a fix actually sticks is in the slides themselves.** Right now you're the only person who can edit the deck — that's the *only* reason it has to be you. (If you'd like, you can give us permission to edit the deck and we'll do all of this for you — see the bottom of the page.)

## How to fix a link in Google Slides (you'll do this same thing for each one)

You'll repeat these exact steps for every fix in Task A:

1. Open your deck in **Google Slides**.
2. Go to the **slide number** the list tells you.
3. Find the **underlined / colored words** it names, and **click once** on them.
4. A little box pops up showing the current web address. Click the **pencil ✏️ ("Edit") button** in that box.
   - *No box appeared?* Instead, drag across the words to highlight them, then press **⌘ + K** (Mac) or **Ctrl + K** (Windows). The same editor opens.
5. **Delete the old address**, and **paste the new one** from the list below (copy it exactly).
6. Click **Apply**.
7. That's it — on to the next one. ✅

> Tip: keep this page open in one window and your slides in another, so you can copy each new address straight across.
